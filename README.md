# 🌱 Spring Framework Learning Journey

## 📌 Overview
This repository contains my learning progress of the **Spring Framework ecosystem**, starting from the basics and gradually moving toward advanced concepts and real-world application development.

I am following a structured roadmap to build strong backend development skills using Java and Spring.

---

## 🚀 Learning Roadmap

I am learning Spring in the following order:

### 1️⃣ Spring Core
- Dependency Injection (DI)
- Inversion of Control (IOC)
- Bean lifecycle
- XML and Annotation-based configuration

### 2️⃣ Spring JDBC
- JDBC Template
- Database connectivity using Spring
- CRUD operations
- Exception handling in JDBC

### 3️⃣ Spring MVC
- MVC architecture
- Controllers and Views
- Form handling
- Request/Response mapping

### 4️⃣ Spring Boot
- Auto configuration
- REST APIs
- Embedded servers (Tomcat)
- Spring Boot project setup

### 5️⃣ Spring Security
- Authentication and Authorization
- Role-based access control
- Password encryption
- Securing REST APIs

### 6️⃣ Spring AOP (Aspect Oriented Programming)
- Cross-cutting concerns
- Logging and transactions
- Aspects, Advices, Pointcuts

---

## 🎯 Goal
To become proficient in building **scalable, secure, and production-ready backend applications** using the Spring ecosystem.
